# Tasklist
Colorful and Animated To-Do List A modern web application to organize tasks, featuring colorful blocks, a flexible horizontal layout, buttons to complete or delete tasks, and smooth animations. All tasks are saved in tasks.json, ensuring persistence even after page reloads. Perfect for a web development portfolio.
README Português Lista de Tarefas Colorida com Node.js Este é um projeto simples de lista de tarefas (To-Do List) desenvolvido com Node.js, Express, HTML, CSS e JavaScript. Ele possui: Layout horizontal com blocos coloridos que mudam de cor a cada tarefa. Botões para concluir e excluir tarefas. Armazenamento de tarefas em arquivo tasks.json. Animações suaves: fade-in, fade-out e hover nos blocos e botões. Validação de input para impedir tarefas vazias. Funcionalidades Adicionar Tarefas: O usuário digita a tarefa e adiciona ao clicar no botão “Adicionar”. Concluir Tarefas: Cada tarefa pode ser marcada como concluída clicando no botão “Concluir”. Excluir Tarefas: Remover tarefas clicando no botão “Excluir”. Cores Aleatórias: Cada tarefa recebe uma cor aleatória e mantém essa cor mesmo após atualizar a página. Persistência: Todas as tarefas são salvas em tasks.json. Features include: Horizontal layout with colorful blocks that change color for each task. Buttons to complete and delete tasks. Task storage in a tasks.json file. Smooth animations: fade-in, fade-out, and hover effects on blocks and buttons. Input validation to prevent empty tasks. Features Add Tasks: Users can type a task and add it by clicking the “Add” button. Complete Tasks: Each task can be marked as completed by clicking the “Complete” button.
